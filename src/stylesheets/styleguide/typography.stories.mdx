# Typography

## Typeface
Minds uses the [Roboto Fontface](https://fonts.google.com/specimen/Roboto) throughout its web and mobile applications. Only the following font weights are used.

* Normal (400)
* Medium (500)
* Bold (700)
* Black (900)

## UI Type scale

### Base
16px is the base size of body copy for UI typography within Minds.

The following font size conventions are currently being used to better standardise the design and development processes, as well as create a cleaner and more consistent user experience. The font sizes increase at consistent set intervals, and each font-size will have its own corresponding line-height.
